# 💰 Finance API (ASP.NET Core + EF Core + SQLite)

API desenvolvida com **ASP.NET Core** com o objetivo de **aprender .NET na prática**, simulando um sistema de gerenciamento financeiro pessoal.

---

## 🚀 Objetivo

Construir uma API REST que permita:

- Registrar transações financeiras (ganhos e gastos)
- Listar e organizar transações
- Calcular saldo atual
- Separar totais de entradas e saídas
- Trabalhar com datas futuras (transações agendadas)

---

## 🧱 Tecnologias utilizadas

- .NET (ASP.NET Core Web API)
- Entity Framework Core
- SQLite
- Swagger

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (OOP)
- Code First com Entity Framework
- Migrations
- CRUD completo
- Organização em camadas

---

## 💰 Funcionalidades

### ✅ Transações
- Criar transações (entrada ou saída)
- Listar todas as transações
- Ordenar por data
- Filtrar por tipo (Income / Expense)
- Suporte a transações futuras

---

### 📊 Cálculos financeiros

- Total de entradas
- Total de saídas
- Saldo atual

> O saldo não é armazenado no banco — ele é calculado com base nas transações.
